Interaction Notes

Behavior specifications for development

Expand/Collapse

Click parent item to toggle expand state. Chevron rotates 180° with 0.3s ease transition. max-height animates for smooth reveal.

Single Selection

Only ONE suboption per category. Clicking a new option deselects the previous one (radio-button behavior).

Type My Own

When "Type My Own" is selected, inline text input appears below options. Input auto-focuses. Value updates in real-time.

Additional Column

Multi-select behavior (checkboxes). Multiple items can be selected simultaneously.

Hover Animation

All items translate 4px right on hover with 0.2s transition. Creates subtle interactive feedback.
